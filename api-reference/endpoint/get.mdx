---
title: "Convert Audio File"
api: "GET https://api.revocalize.ai/convert"
description: "This endpoint takes a WAV vocal audio file and returns a URL string with the converted vocals audio file."
---

### Request

<ParamField body="audio_file" type="file">
  This is the input WAV vocal audio file.
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the conversion was successful. True if successful, False if not.
</ResponseField>

<ResponseField name="message" type="string">
  This provides additional information about the result of the conversion.
</ResponseField>

<ResponseField name="url" type="string">
  This is the URL for the converted vocal audio file.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request GET 'https://api.revocalize.ai/convert' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <token>' \
--form 'vocal_file=@/path/to/audio/file.wav'
```
</RequestExample> 

<ResponseExample>
```json Response
{
  "success": true,
  "message": "Audio file converted successfully.",
  "url": "https://api.revocalize.ai/converted-files/file1234.wav"
}
```
</ResponseExample>
